Face Modelling

Convolutional neural networks (CNNs) have quickly become the de-facto standard model for analysing images, audio etc. The main reason for their success is the hierarchical structure of CNNs which allows them to learn features that are invariant to translations such as rotations. We mentioned earlier that the main issue with CNNs is the limited size of the current 3D datasets. To overcome this issue, a Convolutional Mesh Autoencoder (CoMA) was introduced. An autoencoder is a neural network that is trained to copy its input to its output. Its main purpose is dimension reduction which is the process of reducing the random variables under consideration during learning. An encoder consists of a function to create multiple hidden layers to analyse the input data. A decoder creates a reconstruction of the input from the hidden layer. This is the main benefit of an autoencoder: the hidden layer is smaller than the input layer, yet it can reconstruct the input. This will greatly compress the input data, hence reducing the parameters that the neural network will need to analyse. The following figure shows a convolutional mesh autoencoder at work:


Convolutional Mesh Autoencoders perform much better than state of the art face models. Experiments and results show that using a CoMA model to represent 3D faces perform 50% better than a PCA[] model used in state of the art face models. The main reason for this is that the implementation of autoencoders mean using 75% fewer parameters than the linear PCA models. Having fewer parameters makes the network easier to train and is still more accurate regarding the reconstruction error of the image.
